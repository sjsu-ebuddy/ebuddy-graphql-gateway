## Application setup steps

### Prerequisites:

- Node &gt;10.x
- Yarn
- MongoDB Compass (for accessing the dev database)

### Code setup

Install dependencies

```
yarn install
```

Check environment variables.

Run Application

```
yarn start:dev
```

### To build and run

Build

```
yarn build
```

Start (Please set NODE_ENV as an environment variable).
```
yarn start
```

### To test

Run jest with coverage

```
yarn test
```
